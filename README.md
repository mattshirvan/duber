# Duber app
this is another clone app less focused on UI/UX and more focus on object relational mapper (ORM)

## Prerequisites
Python 3
Flask
Flask-SQLAlchemy
Flask-Migrate
Flask-Bcrypt

## installation
run 
for python 3
'''
pip3 install -r requirements.txt
'''
for python 2
'''
pip install -r requirements.txt
'''
